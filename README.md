# ☕ Coffee Consumption Dashboard

An interactive power-bi style Excel dashboard analyzing coffee consumption behavior, brand preferences, spending patterns, and lifestyle trends.

## 🎯 Objective
To understand consumer behavior and identify insights useful for marketing strategy and customer segmentation.

 📊 Key Insights
- Morning & night show highest consumption
- Filter coffee & mocha lead preferences
- Home consumption dominates lifestyle patterns
- Health-conscious choices increase with age
- Metro vs Tier cities show different trends

📁 Files Included
- Dashboard.xlsx → Interactive dashboard
- dataset.csv → Source data
- dashboard_preview.mp4 → Dashboard demo vedio

🛠 Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Dynamic KPIs
- Data Visualization
